# Travel Budget Prediction ML Model

A Machine Learning-powered travel budget prediction system that estimates the total travel cost per person based on factors such as destination, season, duration of stay, activities, and preferred time to visit.

The project uses predictive analytics and intelligent recommendation techniques to help users plan trips more effectively and manage travel expenses efficiently.

## Features
- Per-person travel budget prediction
- Destination and season-based cost estimation
- Prediction based on staying days and planned activities
- Interactive Streamlit web interface
- Data preprocessing and model training pipeline
- Smart travel budget estimation using Machine Learning models

## Technologies Used
- Python
- Streamlit
- Transformers
- Linear Regression
- NumPy
- Pandas
- Scikit-learn

## Purpose
This project was developed to demonstrate the practical application of Machine Learning and AI in the travel industry by creating an intelligent system that predicts personalized travel expenses for users.
